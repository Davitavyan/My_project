----Комментарии к использовании скриптов----
Для запуска скриптов нужно юзать команду
python name_of_script.py
в терминале.
-----------------------------------------------------------------------
До запуска скриптов нужно установить пакети, если они не установлены.
Например, если вначале кода есть 
import pandas
до прогонки кода нужно в терминале написать код и устанаовить package
pip install pandas
